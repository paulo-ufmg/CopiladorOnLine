Bioclass Python - Bioinformática




Este é o resultado de um trabalho dos alunos da disciplina de Ambientes de Computação da Pós Graduação do Departamento de Ciência da Computação da UFMG, 2024.

Código fonte: Github

A biblioteca Bioclass, tem o objetivo didádico para construção de scripts para um conjunto de códigos de software destinadas a aplicações em bioinformática. Seu principal objetivo é fornecer uma introdução a bioinformática com suas análise de dados biológicos e a resolução de problemas comuns, tais como a manipulação de sequências de DNA, RNA e proteínas

Exemplo de Uso

A forma básica para usar a ferramenta Bioclass para ler um arquivo FASTA e calcular a frequência de nucleotídeos em uma sequenca de DNA:. 

                  # importa a classe Bioprof da biblioteca Bioclass
                  from Bioclass import Bioprof

                  # Instancia o objeto seq com a classe Bioprof
                  seq = Bioprof()

                  # Carrega sequências de um arquivo FASTA e armazena todas em sua biblioteca
                  seq.leiaArquivoFasta(nome_do_arquivo_fasta) 

                  # Os método da Classe Bioprof recebe como argumento uma string com o id da sequência ou um int (index da sequencia armazenada)
                  print(f"Frequência de nucleotídeo: {seq.get_composicao_seq(id_da_sequenicia)}")
Conclusão

"A Bioclass é uma biblioteca para efeitos didáticos para iniciantes em bioinformática para demonstrar como desenvolver uma programação utilizando o paradigma orientada a objeto para construção de ferramentas que facilita o trabalho com dados biológicos em Python. Ela permite introduzir conceitos básico para construção de códigos para análises complexas de maneira eficiente e rápida, contribuindo significativamente para construção de códigos para avanços na pesquisa biológica e médica.."
